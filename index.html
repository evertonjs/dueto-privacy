Dueto

Política de Privacidade

O Dueto é um aplicativo desenvolvido para casais que desejam organizar suas vidas em conjunto — compartilhando tarefas, gastos e eventos de forma simples e privada. Esta Política de Privacidade descreve como coletamos, usamos e protegemos seus dados.

1. Dados Coletados

Ao utilizar o Dueto, podemos coletar as seguintes informações:

  • E-mail: utilizado para criar e acessar sua conta.
  • Nome: utilizado para personalizar sua experiência no app.
  • Foto de perfil: opcional, carregada pelo próprio usuário.
  • Dados financeiros: registros de gastos inseridos voluntariamente pelo usuário (descrição, valor, categoria, data).
  • Tarefas e eventos: informações criadas pelo usuário para uso pessoal ou compartilhado com o parceiro(a).
  • Dados do casal: vínculo entre dois usuários que optam por compartilhar dados dentro do app.

2. Como Usamos os Dados

  • Autenticar e identificar o usuário no aplicativo.
  • Sincronizar dados entre os dispositivos do casal.
  • Exibir resumos e históricos financeiros para o próprio usuário.
  • Personalizar a experiência com o nome e foto de perfil.

Não utilizamos seus dados para publicidade, análises de comportamento em massa ou qualquer finalidade além das listadas acima.

3. Armazenamento e Segurança

Todos os dados são armazenados de forma segura na plataforma Supabase (infraestrutura AWS), com comunicação criptografada via HTTPS/TLS. O acesso aos dados é controlado por políticas de segurança por linha (Row Level Security), garantindo que cada usuário acesse somente seus próprios dados ou os dados do casal vinculado.

4. Compartilhamento de Dados

Não vendemos, alugamos nem compartilhamos seus dados com terceiros. Os dados do casal são visíveis apenas aos dois membros do casal vinculado. Dados marcados como "Particulares" dentro do app são visíveis apenas ao usuário que os criou.

A única exceção é o uso da infraestrutura da Supabase (processamento técnico), que segue os termos de privacidade próprios disponíveis em supabase.com/privacy.

5. Retenção de Dados

Seus dados são mantidos enquanto sua conta estiver ativa. Caso deseje excluir sua conta e todos os dados associados, entre em contato pelo e-mail informado na Seção 7. Processaremos o pedido em até 30 dias.

6. Direitos do Usuário

  • Acessar os dados registrados na sua conta a qualquer momento pelo app.
  • Editar ou excluir qualquer informação diretamente pelo app.
  • Solicitar a exclusão completa da conta e todos os dados por e-mail.
  • Revogar o vínculo com o parceiro(a) a qualquer momento.

7. Contato

Para dúvidas, solicitações de exclusão de dados ou qualquer questão relacionada à privacidade, entre em contato:

E-mail: contato@dueto.app

8. Alterações nesta Política

Esta política pode ser atualizada periodicamente. Em caso de alterações relevantes, notificaremos os usuários pelo aplicativo. O uso contínuo do Dueto após a publicação de alterações implica na aceitação da nova versão.
